# 地点选项讨论清单

这个文件用于 actual page 之前的地点讨论。当前先列分类和判断标准，不急着锁死最终名单。

## 需要讨论的分类

### 丁蜀 / 紫砂 / 陶艺

候选方向：

- 蜀山古南街。
- 中国宜兴陶瓷博物馆。
- 陶二厂相关空间。
- UCCA Clay。
- 紫砂手作体验或工作室探访。
- 本地茶器店或小型展厅。

需要判断：

- 哪些地方更适合第一次理解宜兴。
- 哪些地方只是拍照好看但内容薄。
- 是否安排一次手作体验。
- 是否安排买壶 / 看壶，但不把购物作为主线。

### 春日 / 阳羡茶 / 山野

候选方向：

- 阳羡茶园。
- 茶文化体验。
- 茶室。
- 山间咖啡或轻餐。
- 春日散步点。

需要判断：

- 这条线是否放在上午。
- 是否需要预约茶体验。
- 茶园和丁蜀能否安排在同一天。
- 哪些点交通成本过高。

### 竹 / 洞 / 水

候选方向：

- 宜兴竹海。
- 善卷洞。
- 龙池山。
- 东氿或湖边散步。
- 轻徒步 / 骑行。

需要判断：

- 四个人是否都愿意走路。
- 竹海和善卷洞是否二选一。
- 洞穴类景点是否符合大家兴趣。
- 水边散步更适合白天还是晚上。

### 餐饮 / 夜晚

候选方向：

- 宜帮菜。
- 山珍湖鲜。
- 本地面馆或小吃。
- 茶室。
- 晚上可聊天的空间。

需要判断：

- 需要正式晚餐还是松弛小店。
- 是否需要考虑女生友好、拍照、停车、排队。
- 是否安排夜茶。

### 自由时间 / 打牌

候选方向：

- Saboteur / 打牌。
- 午后避暑休息。
- 茶室坐着聊天。
- 住宿处自由时间。
- 临时空档。

需要判断：

- 哪些时间段应该故意留白。
- 哪些地点附近适合坐下来打牌。
- 自由时间是否进入交通图；默认不进入。
- 是否需要在 UI 里给“不要排太满”的提示。

### 住宿

候选方向：

- 城区酒店。
- 东氿 / 湖边酒店。
- 丁蜀附近住宿。
- 阳羡 / 山里民宿。

需要判断：

- 两男两女的房型安排。
- 是否开车。
- 是否优先交通方便。
- 是否需要风格感。
- 住宿是否成为旅行体验的一部分，还是只是睡觉。

## 筛选原则

- 每天不要超过一个大跨度移动。
- 丁蜀线至少保留一个真正能接触手艺的点。
- 春日线要有茶或山野，不只是拍照。
- 竹洞水线不要贪多，优先保证体力和节奏。
- 住宿地点要参与动线判断，不最后才选。
- 每个模块都要能回答“它让我们进入了宜兴的哪一种感官？”。
- 两个点之间只有确认存在直接交通时才进入网络图连线。
- 如果 A 到 C 应经过 B，就记录 A-B 和 B-C，不记录 A-C。
- 自由时间、休息、打牌是合法模块，不要把每天排满。
