# 体验概念与视觉方向

## 一句话内核

我们去宜兴，不是为了看一只壶，而是为了看一块泥如何变成器，一杯茶如何变成时间，一座小城如何把手艺留在日常里。

## 感官菜单

网站可以像一张可交互菜单，把宜兴拆成几组感官体验。用户 hover 或选择不同区域时，页面色彩、背景、局部动效和信息密度发生变化。

### 1. 丁蜀：泥、火、器

- 关键词：紫砂、陶、窑、工坊、老街、手艺。
- 视觉：红、陶土、暖橙、暗红、窑火高光。
- 气质：扎实、手作、热、慢、近距离。
- 适合内容：蜀山古南街、陶二厂、紫砂体验、陶瓷博物馆、UCCA Clay、工作室探访。
- 交互感觉：hover 时页面像靠近火光，色温升高，按钮或边线出现陶土色 flare。

### 2. 春日：茶、田野、轻风

- 关键词：阳羡茶、茶园、春光、山坡、晨间。
- 视觉：绿、黄绿、浅金、柔白。
- 气质：轻、清、明亮、舒展。
- 适合内容：阳羡茶园、茶文化体验、早午餐、山间散步。
- 交互感觉：hover 时页面变亮，背景更清透，文字间距和留白略放松。

### 3. 竹洞水：竹、洞、湖、湿润空气

- 关键词：竹海、善卷洞、龙池山、东氿、水边、清凉。
- 视觉：绿、蓝绿、深青、雾白。
- 气质：安静、湿润、纵深、降温。
- 适合内容：宜兴竹海、善卷洞、龙池山、湖边散步、骑行。
- 交互感觉：hover 时页面降温，背景色偏蓝绿，过渡更缓。

### 4. 夜晚：饭局、住宿、散步

- 关键词：本地菜、夜茶、民宿、酒店、朋友聊天。
- 视觉：暖白、低饱和棕、深灰、灯光黄。
- 气质：松弛、亲密、收束。
- 适合内容：晚餐、酒吧/茶室、住宿、夜间散步。
- 交互感觉：hover 时降低对比度，界面从探索模式切换到安排模式。

## 页面气质

- 字体：现代 sans-serif，中文优先考虑系统黑体栈。
- 版面：简约，强留白，但不要像品牌官网 landing page。
- 信息结构：左侧或顶部是体验菜单，中间是地点/模块，右侧或下方是行程拼配结果。
- 色彩变化：由 CSS variables 控制，不为每个区域写完全不同的页面。
- 动效：短、轻、克制，用于提示“当前进入哪种感官模式”。

## 不做什么

- 不做纯景点攻略站。
- 不做复杂多页面旅游平台。
- 不把紫砂简化为购物。
- 不把每个地点都做成同样的旅游卡片。
- 不在当前阶段写死最终行程。
