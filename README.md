# Travel

This repository is a static travel workspace intended to be hosted at `travel.raymondfonghow.com`.

Each destination should live in its own folder so it can be hosted as a path under the same domain.

## Destinations

- `yixing/` -> `travel.raymondfonghow.com/yixing`

## Hosting Model

- GitHub repo: `RaymondFongHow/travel`
- GitHub Pages domain: `travel.raymondfonghow.com`
- Destination pages should use relative paths so they work under subpaths like `/yixing`.
- Shared travel-wide assets can be added later at the repo root if useful.

## Current Status

The Yixing project is still in planning. It currently contains Markdown planning docs for a future static card-based itinerary builder.
